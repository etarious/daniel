Tags and their types

Before I go any further, let me tell you of the different types of tags
HTML works with tags "<" and ">", all the elements are created with tags. and there are different types of tags, which are;

  • opened tags
  • closed tags

Opened Tags:

These are tags that needs to be closed when opened, for example the "<html>" tag at the beginning of file just after the doctype declaration is an opened tag. If you observe it was opened as written above as "<html>", and at the end of this file you can it closing as this "</html>".

Closed Tags:

These are tags that does not need a closing tag, they are self closed. a clear example is the "<br>" element, which is sometimes written as "<br />".

Nesting

Elements can be written inside other elements, and this is called nesting.
As you go through this notes on you VS code, you can realize that I put some tags and elements inside some other elements.

The html tag

Looking at the HTML tag just below the doctype declaration, you can observe everyother thing was written inside it, and it consists of 2 sub elements, which are the head element and the body element.
The head elements consists data that are meant for the browser and not for the user, the head element holds some other important elements like the meta tags, title tags, and so on. While anything youe write inside the body element displays on the screen of the browser.
